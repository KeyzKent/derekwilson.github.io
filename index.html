wrist-spin

wrist-spin is a live cricket score display for Pebble watches. Its designed to keep you up to date with your favorite team without having to get your phone out all the time. The source is hosted on bitbucket feel free to download or just browse the code.

Recommender

Recommender is an Android application to store, organize and share personal recommendations. Its designed to be used in person where you want to pass on a recommendation to a friend. There is no web service, there is no need to register you details anywhere. The source is hosted on bitbucket feel free to download or just browse the code.

SendRecommendation

SendRecommendation is an Windows 10 Universal application to send recommendations to the main Recommender application. The source is hosted on bitbucket feel free to download or just browse the code.

RamEater

RamEater is a memory stress tool for Android devices. The source is hosted on github feel free to download it or just browse the code.

MeasureMe

MeasureMe is a low friction personal measurement tool for Android devices. The source is hosted on bitbucket feel free to download it or just browse the code. There are also example spreadsheets available to download

Podcast Utilities

Podcast Utilities is an open source podcast downloader application and api for the .NET Framework. Its hosted on bitbucket feel free to download it or just browse the code.

TextByNumbers

TextByNumbers is an open source text snippet assembler for SMS and emails. Its available as an HTML5 offline application and as a native android application from Google Play or download the APK.

Worldolio

Worldolio is a shareware windows desktop, windows mobile and web application providing timezone and other geographical information.
